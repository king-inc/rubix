<H3><b>Abbreviation meanings:</b></H3>
"L" = Left <br>
"LU" = Left Line Up <br>
"LD" = Left Line Down <br>
"R" = Right <br>
"RU" = Right Line Up <br>
"RD" = Right Line Down <br>
"U" = Up <br>
"UL" Up Line Left <br>
"UR" = Up Line Right <br>
"D" = Down <br>
"DL" = Down Line Left <br>
"DR" = Down Line Right <br>
